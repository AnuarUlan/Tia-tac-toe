# Tia-tac-toe
#mcpi #python 
это игра на основе крестиков ноликов
# Структура проекта
1. server — Директория с ядром локального сервера Minecraft. Содержит предустановленные конфигурации и программные интерфейсы для связи с Python.
1. Start_Server.lnk — Файл для быстрого запуска серверной части.
1. mcpi — Локальная библиотека API для передачи команд от скрипта к игровому миру.
1. race.py — Основной программный файл, содержащий игровую логику, систему обработки коллизий и генерацию уровней.
1. input_system.py — Модуль для обработки нажатий клавиш и управления транспортным средством.
1. requirements.txt — Список внешних зависимостей Python.
1. LICENSE — Текст лицензионного соглашения.
# Какие def я использовал
1. Для крестов и ноликов я использовал def draw который брал картинки из tictactoe.py
2. Еще я использовал def __init__ как родительский
# Требования
1. Крестики и Нолики подерживаються на версий 16.5
# Харектиристика игры:
1. Сетка три на три 
2. Обводка из алмазов и клетки из шерсти
# Механики 
1. Что бы поставить крестик или нолик надо встать над клеткой и нажать левую кнопку мышки
2. Что бы победить надо сделать вертикально или горизантально в ряд крестики или нолики
